# Product Demo Web

Angular web application for the Product Management Demo.

## Suggested Repository Name

`product-demo-web`

## Tech Stack

- Angular 15.2.10
- TypeScript 4.8.4
- Bootstrap 5.2.3
- Reactive Forms
- Angular Router
- Browser `localStorage`

## Node Compatibility

- Node.js `14.21.3`
- npm `6.14.18`

## Install

```bash
nvm use
npm install
```

## Run

```bash
npm start
```

Application URL:

`http://127.0.0.1:4200/products`

## Build

```bash
npm run build
```

## Unit Test

```bash
npm test
```

## Notes

- The app initializes mock data into `localStorage` with key `product-demo-products`
- The app includes list, detail, add, edit, delete, search, filter, sort, pagination, and reset mock data
